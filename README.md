Afin de lancer le projet vous devez ;

DANS LE DOSSIER BACKEND :
------------------------

Télécharger les modules :
- npm install

Démarrer l'API :
- npm run dev


DANS LE DOSSIER FRONTEND/GROUPOMANIA :
-------------------------------------

Télécharger les modules :
- npm install

Démarrer le serveur frontend :
- npm run serve

(README dans le dossier "frontend")
